Voice Launcher

This program requires:
 
Windows 7

Microsoft .Net Framework 
http://www.microsoft.com/net

and the Speech SDK
http://www.microsoft.com/en-us/download/details.aspx?id=10121

only the SpeechSDK51.exe is needed


There are a few features that are currently being worked on:

	Autosave to settings
	Autoload previous settings
	Search feature using dictation
	Home automation features like temperature and such
	Arduino integration

Warning:  This program has not been fully tested on multiple machines.  It works on mine.
